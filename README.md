# Running Code Surfer

```bash
#install dependencies
yarn

# run code surfer on  http://localhost:8000/
yarn start

# build
yarn build
```

# Running Example Projects
## Composition API Project
```bash
# navigate to folder
cd project/composition

#install dependencies
yarn

# run project on  http://localhost:8080/
yarn serve
```

## Options API Project
```bash
# navigate to folder
cd project/options

#install dependencies
yarn

# run project on  http://localhost:8080/
yarn serve
```
